# 什么是 HuoShan UI？

HuoShan UI 是一个现代化的无头组件库，专注于提供高质量的组件逻辑而不附带预设样式，让开发者能够完全掌控组件的外观。

## 特性

### 🎨 无头设计
- 组件只提供功能逻辑，没有预设样式
- 完全可自定义的外观
- 更大的灵活性和创造空间

### ⚡️ 高性能
- 基于 Vue 3 开发
- 最小化的运行时开销
- 支持树摇优化

### 🛠️ 类型支持
- 使用 TypeScript 编写
- 完整的类型定义
- 更好的开发体验

### 🔌 框架支持
- 支持 Vue 3
- React 版本开发中
- 未来会支持更多框架

## 为什么选择 HuoShan UI？

### 1. 更大的自由度
传统组件库通常会带有预设样式，这些样式有时候会与项目的设计风格产生冲突。HuoShan UI 采用无头设计，让你可以完全掌控组件的外观，轻松适应各种设计需求。

### 2. 更好的性能
我们在开发时特别注重性能优化，确保组件在运行时具有最小的开销。同时支持树摇优化，只打包你使用的功能。

### 3. 更强的类型支持
完整的 TypeScript 类型定义让你在开发时能够获得更好的类型提示和错误检查，提高开发效率。

### 4. 更多的扩展性
组件的核心逻辑与UI展示分离，这使得组件更容易扩展和定制。你可以基于相同的逻辑创建多个不同风格的组件。

## 哪些场景适合使用？

HuoShan UI 特别适合以下场景：

- 需要完全自定义组件样式的项目
- 对性能有较高要求的应用
- 需要统一管理多个品牌主题的系统
- 需要频繁更换UI主题的项目
- 想要完全控制组件行为的场景
